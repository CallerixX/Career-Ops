# Career-Ops

Локальная веб-платформа для автоматизации поиска работы. Система позволяет извлекать описания вакансий по ссылке, анализировать их с помощью ИИ, автоматически адаптировать ваше резюме под требования компании и вести учет всех откликов на встроенной канбан-доске.

## Функционал

- Обход блокировок и парсинг: автоматическое извлечение текста вакансии. Для hh.ru реализован механизм обхода защиты через прямое использование JSON API.
- Глубокий анализ релевантности: ИИ сопоставляет ваше резюме с текстом вакансии, оценивает совпадение по ключевым метрикам (технологический стек, опыт, корпоративная культура) и формирует SWOT-отчет.
- Адаптация резюме под ATS: создание кастомизированной версии резюме, содержащего необходимые ключевые слова. Доступен экспорт документа в строгом формате PDF для отправки работодателю.
- Трекинг откликов: канбан-доска для отслеживания стадий прохождения собеседований (Отправлено, Интервью, Оффер, Отказ).

## Технический стек

- Фреймворк: Next.js (App Router)
- База данных: SQLite + Drizzle ORM
- ИИ интеграции: Vercel AI SDK (поддержка OpenRouter и Gemini)
- Генерация PDF: Playwright
- Стилизация: Tailwind CSS

## Установка и запуск

1. Склонируйте репозиторий на свой компьютер:
git clone https://github.com/CallerixX/Career-Ops.git

2. Перейдите в папку проекта и установите зависимости:
npm install

3. Установите браузеры, необходимые для работы Playwright (генерация PDF):
npm run playwright:install

4. Создайте файл окружения: скопируйте .env.local.example в .env.local и укажите свои ключи API (например, токен OpenRouter).

5. Инициализируйте локальную базу данных SQLite:
npm run db:push

6. Запустите сервер разработки:
npm run dev

## Подготовка к работе

После запуска системы перейдите в раздел интерфейса "Настройки". Вставьте свой базовый профиль и резюме в формате Markdown (с описанием опыта, навыков и образования). Эти данные будут использоваться платформой как исходные для сравнения с вакансиями.
